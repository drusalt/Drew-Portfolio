Awesome HTML and CSS links that have helped me along the way!

Validator W3

Markup Validator Service

This website allows you to upload your HTML files, run them through a code validator and fix any errors that show up. It's an extermely useful tool for writing and checking your HTML code. The errors are sometimes a little confusing, but it almost always points you in the direction of where to start.

W3schools Website

HTML Color Picker

This website allows you to go through any existing color in the color realm and link it to your website. Since websites read either basic names like, red, blue or green AND hashtag values for everything else that is a blend of colors, you can easily find the perfect hashtag color for your website!

Fonts from Google

Google Fonts

Google fonts allows you to link most of the popular fonts to your website for free. It has hundreds to choose from and offers a really nice interface which allows you to copy and link the font in your HTML code. You can link different sizes of font as well as different iterations of bold.
